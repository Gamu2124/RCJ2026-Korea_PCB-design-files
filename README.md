# RCJ2026-Korea_PCB-design-files

This repository contains the PCB design files for RoboCup Junior 2026.

### Overview

This repository provides the PCB design files used in the RoboCup Junior 2026 competition.

The files are available for reference, manufacturing, modification, and educational purposes.

### Contents

The repository may include:

* KiCad project files (`.kicad_pro`)
* Schematic files (`.kicad_sch`)
* PCB layout files (`.kicad_pcb`)
* Manufacturing files (Gerber, Drill, etc., if available)
* Custom footprints and symbol libraries (if required)

### Software

These PCB design files were created using **KiCad**.

For the best compatibility, it is recommended to use the same major version of KiCad that was used to create the project.

### Notes

* These files are provided for reference and educational purposes.
* Manufacturing or modifying the boards is done at your own responsibility.
* Some electronic components may require suitable alternatives depending on availability.
